# broyouwantart.com

Reference-inspired static portfolio for Eugene Lim / Bro You Want Art?.

## Direction

Built from these references without copying their content:

- tomisinanimashaun.com — sparse red micro-navigation and asymmetrical image-led portfolio energy.
- galerie5b.net — split catalogue/information layout, tiny captions, quiet gallery/publishing tone.
- sam-evers.com — white space, serif navigation, standalone images as objects rather than cards.
